## How to Add Your Gym Equipment List

1. **Place your Excel file here** in this `data` folder
2. **Name it exactly**: `gym_equipment.xlsx`
3. **File format**: Excel file (.xlsx or .xls)

## Required Excel Columns

Your Excel file should have these columns:
- **Equipment** - Name of the equipment (e.g., "Bench Press", "Squat Rack")
- **Quantity** - Number available (e.g., 2, 1, 10)
- **Location** - Where it's located (e.g., "Main Area", "Cardio Zone")
- **Status** - Current status (e.g., "Available", "Maintenance")

## Example

| Equipment | Quantity | Location | Status |
|-----------|----------|----------|--------|
| Bench Press | 2 | Main Area | Available |
| Squat Rack | 1 | Main Area | Available |
| Dumbbells | 10 | Free Weights | Available |
| Treadmill | 3 | Cardio Zone | Available |

## What Happens Next

Once you place your `gym_equipment.xlsx` file here:
- The app will automatically load your equipment list
- AI will create personalized routines based on YOUR specific gym equipment
- You'll see your equipment displayed in the sidebar
- All recommendations will be tailored to what you actually have available

## Current Status

If no file is found, the app will show a sample equipment list as a placeholder.
